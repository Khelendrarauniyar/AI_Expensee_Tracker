About This Project

Welcome to the Expense Tracker web application! This project is designed to help you manage your personal finances by tracking your expenses and categorizing them. With this application, you can:

  • Add and manage expense categories
  • Record your daily expenses
  • View detailed analytics and reports
  • Track your spending habits over time

The Expense Tracker is built using Flask, a lightweight web framework for Python. It uses SQLAlchemy for database management and Jinja2 for templating. The front-end is styled with custom CSS to provide a clean and modern user interface.

We hope this application helps you gain better control over your finances and make informed decisions about your spending.
